# PRODUCT.md

Status: `current`

## Register

**Product.** This is an app UI / personal tool. Design serves the work; it does not perform.

## Product purpose

PromptFlow Library is a single-user "AI command library" — a serious personal system to store, organize, version, reuse, and evaluate the prompts, workflows, notes, and ideas a builder uses across AI tools (Codex, Claude Code, Cursor, ChatGPT, image/video models). It also carries a lightweight capture inbox so ideas don't disappear.

It is explicitly **not** a generic notes app and **not** a commercial SaaS. It is a private operating system for one power user's prompt craft, plus a read-only demo for visitors.

## Users

One person: a solo developer / product builder who lives in AI tools daily. Power user, keyboard-first, values speed and density over hand-holding. Opens the app dozens of times a day to capture or copy something fast. A secondary audience is a casual visitor browsing the read-only demo.

## Scene

A builder at a 27-inch monitor late at night, dim room, mid-flow between editor and terminal, flicking to PromptFlow to grab a reliable prompt or dump an idea before it's gone. The app should feel like a fast instrument panel, not a document.

## Brand & tone

Fast, focused, premium, technical, slightly futuristic. A "dark productivity cockpit." Confident and quiet, not loud. Information-dense but never cluttered. Reads like a tool built by an engineer for themselves.

## Strategic principles

- **Speed to capture / copy is sacred.** The fastest paths (⌘K, quick capture, copy prompt) must always feel instant and obvious.
- **Density with calm.** Show a lot without noise: tight rhythm, clear hierarchy, restrained color.
- **Earn every pixel.** No decoration for its own sake. Accent color marks meaning (category, status, score), never mood.
- **Keyboard-native.** Shortcuts and the command palette are first-class, not afterthoughts.

## Anti-references

- Generic Notion/notes-app blandness (cream surfaces, soft everything).
- SaaS-cliché dashboards: gradient hero numbers, identical icon+heading+text card grids.
- Neon-on-black "AI/crypto" theme reflex.
- Corporate admin-panel grey.
